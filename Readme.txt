Red Bus Project
Author: Yamini Devi

Instructions to run the code:
1.	Run "Selenium_redbus.ipynb" to get the bus route details and their links.

2.	Run "busdetails.ipynb" to get the details of the various details such as 
	Bus_name
	Bus_type
	Start_time
	End_time
	Total_duration
	Price
	Seats_Available
	Ratings
	Route_link
	Route_name

3. Run "sqlconn.ipynb" to store data in SQL database.

4. Run "redbusapp.py" usign the command "streamlit run redbusapp.py" in order to start the app

5. Choose the preffered option in the app.
